This zipped file contains the following files:

*_meta.txt – contains the metadata for the data file, including general information, definitions and footnotes.

*CSV_data.csv – contains the data in a comma delimited format
or
*TAB_data.tab – contains the data in a tab delimited format.


Data file record layout (column headings):

CENSUS_YEAR		int		Census year
DGUID			varchar(19)	Dissemination Geography Unique Identifier
ALT_GEO_CODE		varchar(19)	Alternate geographic code
GEO_LEVEL		varchar(75)	Geographic level
GEO_NAME		varchar(200)	Geographic name
FIRST_NAME		varchar(50)	First name
GENDER_TOTAL_COUNT	int		Total - Gender, Count
GENDER_TOTAL_PERCENTAGE	float		Total - Gender, Percentage
GENDER_TOTAL_RANK	int		Total - Gender, Rank
MEN+_COUNT		int		Men+, Count
MEN+_PERCENTAGE		float		Men+, Percentage
MEN+_RANK		int		Men+, Rank
MEN+_SYMBOL		varchar(5)	Men+, Symbol
WOMEN+_COUNT		int		Women+, Count
WOMEN+_PERCENTAGE	float		Women+, Percentage
WOMEN+_RANK		int		Women+, Rank
WOMEN+_SYMBOL		varchar(5)	Women+, Symbol
